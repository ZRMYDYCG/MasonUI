# Message 信息提示

✅ 即时反馈：无需用户操作，自动触发展示，支持手动关闭

✅ 灵活定位：预设多种全局位置（如顶部居中、底部右侧），消息队列智能堆叠

✅ 类型化设计：内置 Success/Error/Warning/Info 四种语义化样式，图标与颜色自动匹配

✅ 可控生命周期：自由设定持续时间，支持悬停暂停销毁计时

✅ 无障碍支持：默认适配屏幕阅读器
